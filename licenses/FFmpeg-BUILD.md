# FFmpeg 分发说明

便携目录中的 `ffmpeg.exe` 与 `ffprobe.exe` 来自 gyan.dev 的 Windows 64-bit static essentials build，版本为 8.1.1。该构建启用了 GPL 组件（包括 libx264 与 libx265），构建方将其标为 GPL v3。

- 构建说明和完整配置：`FFmpeg-Windows-Build-README.txt`
- GPL v3 文本：`FFmpeg-GPL-3.0.txt`
- 对应 FFmpeg 源码提交：https://github.com/FFmpeg/FFmpeg/commit/239f2c733d
- 构建方主页：https://www.gyan.dev/ffmpeg/builds/
- FFmpeg 法律说明：https://ffmpeg.org/legal.html

H.264/H.265 还可能涉及所在地的软件专利或编解码许可。本说明不是法律意见；公开或商业分发前，应由分发方审查 GPL 对应源码义务、第三方库条款与专利许可。
